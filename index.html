📦 How it works

This is a self-contained moon widget that:

  • Calculates moon phase based on latitude/longitude and current date
  • Renders a visual moon using SVG with accurate illumination
  • Supports full (card) and compact (strip) layouts
  • Auto-initializes on .ac-widget-moon elements
  • No external API calls needed — works offline

Attributes:

  • data-lat + data-lng – location coordinates (required)
  • place – display name (optional)
  • data-size – "full" or "compact"

✨ The widget automatically finds all .ac-widget-moon elements on the page.
